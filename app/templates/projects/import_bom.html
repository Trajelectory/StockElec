📋 Formats supportés

StockElec reconnaît automatiquement les BOM exportées depuis :

  • KiCad 7 / 8 — via Tools → Edit Symbol Fields → Export
  • KiCad + plugin JLCPCB — colonnes Comment, Designator, Footprint, LCSC Part Number
  • KiCad + bom2csv — séparateur ; supporté

La colonne LCSC reconnue peut s'appeler :

  • LCSC Part Number
  • LCSC Part #
  • LCSC
  • Supplier Part Number

💡 Assure-toi d'avoir renseigné le champ LCSC sur tes composants dans KiCad pour que la comparaison fonctionne.
